Esta utilidad propietaria de Biblioteca Arrullín permite generar carnets bibliotecarios rápidamente introduciendo los datos de el cliente.
Esta página web es una ayuda para las posibles dudas que puedan surgir a la hora de utilizar este software.

Funcionalidades principales

  • Inicio de sesión:
    El administrador de la biblioteca inicia sesion con sus credenciales de administrador
  • Generación de carnets:
    Permite crear carnets bibliotecarios personalizados para los usuarios, incluyendo foto y datos personales.
  • Impresión y exportación de carnets:
    Ofrece opciones para imprimir los carnets generados o exportarlos en formatos digitales.
  • Soporte multilingüe:
    La aplicación está disponible en varios idiomas, mejorando la accesibilidad para usuarios de diferentes regiones.

Preguntas Frecuentes

¿Cómo puedo deshacer una acción?

Para deshacer una acción, utiliza el menú "Edición" y selecciona "Deshacer". También puedes usar el atajo de teclado Ctrl+Z.

¿Cómo puedo rehacer una acción?

Para rehacer una acción, utiliza el menú "Edición" y selecciona "Rehacer". También puedes usar el atajo de teclado Ctrl+Y.

¿Cómo puedo reiniciar los datos del formulario?

Para reiniciar los datos del formulario, selecciona la opción "Nuevo Carnet" en el menú "Archivo". Esto limpiará todos los campos del formulario para que puedas empezar de nuevo.

¿Cómo puedo guardar el carnet en un archivo CSV?

Para guardar el carnet en un archivo CSV, utiliza la opción "Guardar Datos" en el menú "Archivo". Esto exportará los datos del carnet a un archivo CSV que podrás abrir con cualquier editor de hojas de cálculo.
